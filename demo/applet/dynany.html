This is the dynany-Example. Watch your JAVA console to see the output. To make this example run you have to:
  • start the JacORB NameServer (ns ior-file)
  • start the JacORB Appligator (appligator ior-file)
  • start the DII-Server (jaco demo.dynany.Server)
  • make sure the jacorb.properties file in $JACORB_HOME/demo/applet has the right settings
  • use JacORB's CORBA classes instead of Netscape's (see the Programming Guide )
