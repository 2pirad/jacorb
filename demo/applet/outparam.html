This is the OutParam-Example. Watch your JAVA console to see the output. To make this example run you have to:
  • start the JacORB NameServer (ns ior-file)
  • start the JacORB Appligator (appligator ior-file)
  • start the OutParam-Server (jaco demo.OutParam.dynamicServer)
  • make sure the jacorb.properties file in jacorb/demo/applet has the right settings
  • use JacORB's CORBA classes instead of Netscape's (see README)
