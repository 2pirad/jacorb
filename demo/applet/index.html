Appligator

To make the following examples work you have to start the naming service (ns) and the corresponding servers first. They are located in the $JACORB_HOME/demo/<example> directories. Available examples are dii, mtclient, dynany and outparam.

Of course you need the Appligator running, too (see the Programming Guide). Finally you have to adapt your jacorb.properties file and place it in this directory ($JACORB_HOME/demo/applet).

Text only examples (you need to have your JAVA Console open):

  • DII Example
  • Outparameter Example
  • MultiThreaded Client Example
  • Dynamic Any Example
